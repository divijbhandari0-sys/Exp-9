Aim:

Tools for EDA- Study Of Numpy Library

Theory

NumPy is a Python library used for efficient numerical computations with multi-dimensional arrays.

In this code:

Arrays are created using np.array() in both 1D and 2D forms.

Array attributes such as ndim, size, shape, and dtype are used to understand structure and data type.

Built-in functions like np.zeros(), np.ones(), np.eye(), np.arange(), and np.linspace() are used to generate arrays quickly.

Element-wise arithmetic operations (e.g., addition and multiplication) are performed without loops.

Statistical functions such as np.mean(), np.median(), np.max(), np.min(), and np.sum() are used to analyze numerical data.

NumPy improves performance and simplifies mathematical operations compared to standard Python lists.

Conclusion

This project demonstrates the basic features of NumPy, including array creation, property inspection, mathematical operations, and statistical analysis. It provides a strong foundation for numerical computing and prepares the learner for advanced applications in data science, machine learning, and scientific computing.
